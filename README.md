# Elder Care Backend API

FastAPI backend for an elderly care mobile application  
Built with **FastAPI**, **MSSQL**, **SQLAlchemy Core**, and **Firebase Cloud Messaging**

---

## Tech Stack

- Python 3.1
- FastAPI
- MSSQL (SQLAlchemy Core)
- APScheduler
- Firebase Cloud Messaging (FCM)
- JWT Authentication
- Windows + VS Code
